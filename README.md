# MonkeypoxSEIR
This is simulator of "Monkeypox" in Japan (first parameter settings : S=N-I+E, I+E=60, R=0) use standard SEIR model. 
<p>I used part of formula from [Modeling the Transmission Dynamics of the Monkeypox Virus Infection with Treatment and Vaccination Interventions](https://www.scirp.org/pdf/JAMP_2017121414351920.pdf)</p>
